# 跃进计划介绍
跃进计划，是老池子以及回响项目组共同创作的一个大型项目。

故事背景发生在2100年，地球突如其来的“季节变化异常”造成全球性的经济损失，调查发现是由于人类过量开发宇宙，导致宇宙环境严重混乱。

为了维护宇宙环境安全，地球开启了“跃进计划”，一边改善宇宙环境，一边寻找着新的家园。
